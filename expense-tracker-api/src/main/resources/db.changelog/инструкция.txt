0. Не менять ничего в папках `changesets` и `sql`.

1. Написать sql скрипт на добавление/изменение поля и тд и поместить его в папку `sql`.

2. Добавить НОВЫЙ файл changeset в папку `changesets` - пример есть в `init-tables.xml`
        (надо поменять `id`, `author` и название файла в атрибуте `path`)

3. Добавить этот changeset в `db.changelog-master.xml` - пример там есть уже
        (надо поменять название файла в атрибуте `file`).

4. При запуске этот скрипт должен прогнаться.

Скрипты, которые прогнались можно посмотреть в таблице `databasechangelog`
(Если возникли какие-то проблемы можно удалить таблицы changelog`а - их там 2)
